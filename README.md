# Passive-Terahertz-Images

**Passive-Terahertz-Images** is a passive terahertz (THz) security-screening image dataset for object detection. The dataset contains THz images and bounding-box annotations for three categories: **Pistol**, **Knife**, and **Other**.

## Dataset Summary
- **Modality:** Passive terahertz (THz) security-screening images
- **Task:** Object detection (bounding boxes)
- **Classes:** Pistol, Knife, Other
- **Data:** Images + annotations
- **License:** CC BY-NC 4.0 (see License section)

Download the release package and unzip it locally.


## Annotation Format
Bounding boxes are provided for each image.  

**Class names:**
- `Pistol`
- `Knife`
- `Other`
